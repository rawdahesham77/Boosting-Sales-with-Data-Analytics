📊 ShopEasy Marketing & Customer Insights Dashboard
📌 Project Overview

This project provides a comprehensive analysis of marketing performance and customer sentiment for ShopEasy, using interactive dashboards built in Power BI.

The goal is to transform raw data into actionable insights that help improve marketing strategies, enhance customer experience, and support data-driven decision-making.

🛠 Tools & Technologies
Power BI
Microsoft Excel
Power Query
Data Modeling
Data Visualization
📊 Dashboard 1: Marketing Performance Overview

This dashboard focuses on analyzing marketing campaigns and their effectiveness.

🔹 Key Metrics:
Total Revenue
Conversion Rate
Average Order Value (AOV)
Engagement Rate
🔹 Insights Provided:
Marketing Reach by Campaign
Conversion Rate by Campaign
Engagement Rate by Content Type (Newsletter, Blog, Social Media, Video)
Total Interactions by Campaign
Engagement Trend Over Time
🎯 Purpose:
Identify top-performing campaigns
Analyze customer engagement behavior
Track performance trends over time
💬 Dashboard 2: Customer Voice & Sentiment Analysis

This dashboard analyzes customer feedback and reviews to understand customer satisfaction.

🔹 Key Metrics:
Average Rating
Total Reviews
Average Sentiment Score
🔹 Insights Provided:
Reviews distribution by themes (Product Quality, Shipping, UX, etc.)
Rating distribution (High / Medium / Low)
Sentiment distribution analysis
Customer feedback trends
🎯 Purpose:
Understand customer satisfaction drivers
Identify common issues and pain points
Improve product and service quality
📈 Key Insights
Certain marketing campaigns generate significantly higher reach and engagement
Engagement varies across content types, with some formats performing better than others
Customer feedback is mostly concentrated around product quality and delivery
Sentiment analysis highlights areas needing improvement
